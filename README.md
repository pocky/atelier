# Atelier Forum PHP

## Prérequis

Docker :
- Docker for windows
- Docker for mac
- Docker + docker-compose

Spécifique à windows :
- Make

## Déroulé

- Cloner le repository `git clone git@github.com:pocky/atelier.git`
- `make build`
- `make start`
- Faire l'atelier
- `make stop`
